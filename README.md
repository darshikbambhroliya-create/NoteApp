📝 NoteApp

A simple and modern React Native Notes App with features like add, edit, delete, search notes, and dark mode support.

🚀 Features
➕ Add notes (title + content)
✏️ Edit notes
❌ Delete notes
🔍 Search notes
🌙 Dark mode toggle
🖼️ Image background UI support
📱 Mobile responsive React Native UI
🛠️ Tech Stack
React Native
Expo (if used)
TypeScript / JavaScript
React Hooks (useState)
Bun (package manager)
📦 Installation

Clone the repository:

git clone https://github.com/darshikbambhroliya-create/NoteApp.git
cd NoteApp
⚡ Install dependencies (using Bun)
bun install
▶️ Run the app

If using Expo:

bunx expo start

OR

npx expo start
